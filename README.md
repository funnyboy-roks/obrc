# One Billion Row Challenge

This is Rust implementation of the [One Billion Row Challenge].

[One Billion Row Challenge]: https://github.com/gunnarmorling/1brc

## Run

Generate the `measurements.txt` from [1brc] and place into the current
directory.  Then run using:

[1brc]: https://github.com/gunnarmorling/1brc

```sh
cargo r --release
```
